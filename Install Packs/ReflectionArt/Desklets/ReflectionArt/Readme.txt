Reflection AlbumArt 2 for AveDesk 1.2+
======================================

You are free to re-upload this widget with different
graphics at http://www.avedesk.org.


Needed:
 + iTunes, or
 + WinAmp, with http://www.winamp.com/plugins/details.php?id=143299 genactivewinamp installed. Don't forget to reload winamp after installing the plugin.

Features:
 + shows albumart from iTunes or winamp (with gen_activeamp plugin).
 + Option to AutoHide the widget when none of these players is active
 + Can show the albumart only or placed inside a CD-case
 + A reflection is added to the widget.
