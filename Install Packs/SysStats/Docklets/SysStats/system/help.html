SysStats

 

SysStats is a flexible application that you can use to produce widgets for ObjectDock and AveDesk with no programming. You do this by wiring up a bunch of pre-defined components called Overlays, Meters and Controllers.

 

SysStats comes with some widgets and more are available for download. You can use these as the basis for creating your own, or when you are comfortable with SysStats you can create them from scratch.

 

SysStats provides two ways that power-users can extend its functionality:

 

  • Scripts
  • Plugins

 

You can write meters or controllers using many scripting languages and you can write additional types of overlay, meter or controller using C++.
